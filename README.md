# mini-project-dashboard
This project demonstrates async data handling in React with retries, fallbacks, interceptors, and routing.

# User Dashboard (React)

## Features
- Centralized Axios API layer
- Request & response interceptors
- React async data fetching with useEffect
- Parallel API calls using Promise.all
- Retry mechanism for temporary failures
- Fallback using cached data

## Tech Used
- React
- Axios
- JavaScript (async/await)
- react-router-dom

## Key Learnings
- Handling async operations safely in React
- Improving performance with parallel requests
- Making applications resilient to API failures
